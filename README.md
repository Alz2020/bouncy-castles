# Bouncy Castles Booking System

A Python-based command-line application to manage **bouncy castle rentals**, customers, and bookings using SQLite. This project demonstrates database operations, user input handling, and basic CRUD functionality.

---

##  Overview

**Bouncy Castles** allows users to:

- Manage customer information (name, contact, address)
- Store details of bouncy castles (dimensions, color, capacity, hire price)
- Record bookings (customer, castle, date, notes)
- View, search, insert, and delete data directly from the terminal

It’s a practical demonstration of **Python programming**, **SQLite database management**, and **interactive console applications**.

---

## Features

- **SQLite integration:** Store persistent data for customers, bouncy castles, and bookings  
- **CRUD operations:** Create, Read, Update (via insert/delete), and Delete data  
- **Search functionality:** Search bookings by customer ID, castle ID, or date  
- **Interactive menu:** Easy-to-use console interface for all operations
